# Linux Server Monitoring Scripts

Automated monitoring scripts for a nginx web server on Ubuntu.
Built as part of my Linux and Cloud studies.

## Scripts
- check.nginx.sh - checks if nginx is running and restart it if it crashes
- check-disco.sh - monitors disk usage and alerts if above 80%
- relatorio.sh - generates a full system report

## Technologies
- Linux Ubuntu
- Bash scripting
- nginx
- crontab
